# LEXA Tier Definitions & Upsell Packages
**Purpose**: Complete feature matrix for all tiers and upsell packages  
**Status**: Finalized by User  
**Created**: January 5, 2026

---

## Overview

LEXA offers **3 subscription tiers** and **4 upsell packages** per experience.

### Key Rules:
- ✅ **Higher tiers include all features of lower tiers**
- ✅ **Higher upsell packages include all features of lower packages**
- ✅ **Subscription must be paid yearly** (prevents abuse: user can't get all scripts in first month then cancel)
- ✅ **Limited fully curated scripts per tier** (if user wants more, they can buy on-demand)
- ✅ **On-demand pricing applies** if user exceeds tier limits

## Value Proposition Summary

### Free (The Spark) - €0/year
**Best For**: Trying LEXA, getting 1 basic script per year

**Best For**: Regular travelers who want unlimited basic scripts + 3 fully curated experiences per year
**Value**: 3 fully curated scripts = €4,491 if purchased separately (3 × €1,497 Blueprint), but included in €3,564/year subscription

**Value**: 5 fully curated scripts = €14,985 if purchased separately (5 × €2,997 Concierge), but included in €11,964/year subscription
